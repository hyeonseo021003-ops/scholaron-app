# 장학온 (ScholarOn)

> 흩어진 장학금을 한 곳에서 맞춤으로 — **B2C 장학금 매칭 & 1:1 후원 서비스**

대학생을 위한 장학금 통합 검색·추천과, 소득분위 사각지대를 해소하는 **1:1 매칭 후원**을
하나의 앱에서 제공하는 Flutter 데모 애플리케이션입니다.
(노션 기획 "B2C 장학금 매칭 서비스"를 기반으로 구현)

## 핵심 기능

| 기능 | 설명 |
| --- | --- |
| 🔎 **장학금 맞춤 추천 / 검색·필터** | 키워드 검색 + 유형/소득분위 무관 필터, 마감 임박·사각지대 해소 장학금 우선 추천 |
| 💜 **1:1 매칭 후원** | 피후견인의 **자기소개서·학업계획서**를 보고 후견인이 직접 후원을 결정 (선배→후배, 교수→제자) |
| 🧾 **세액공제 / 후원 증빙** | 후원 금액 기반 예상 세액공제액 산출 및 기부금 영수증 발급(데모) |
| 🙋 **마이페이지** | 후원 현황, 누적 후원액, 후원 중인 학생 수 |

## 기획 배경 (요약)

- 장학금 정보가 산재되어 조건에 맞는 장학금을 찾기 어려움 → **통합 검색·추천**
- 소득분위로 인해 지원이 필요한 학생도 혜택을 받지 못하는 사각지대 → **분위 무관 매칭 후원**
- 분야별 상이한 활동 지원 격차 → **과/동아리 단위 후원으로 확장 가능한 매칭 구조**

## 프로젝트 구조

```
lib/
├── main.dart                # 앱 진입점, Provider/테마 설정
├── theme/app_theme.dart     # Material 3 브랜드 테마, 한글 폰트
├── models/                  # Scholarship / Mentee / Sponsorship 모델
├── data/mock_data.dart      # 데모용 목업 데이터
├── state/app_state.dart     # 검색·필터·후원 상태 (ChangeNotifier)
├── widgets/                 # ScholarshipCard, MenteeCard
├── screens/                 # 목록·상세·매칭·후원현황·마이 화면 + 루트 셸
└── utils/format.dart        # 통화/마감일 포맷
```

## 실행 방법

```bash
flutter pub get

# 모바일/데스크톱
flutter run

# 웹 (CDN 미의존 빌드 — 폐쇄망/오프라인에서도 한글 정상 렌더링)
flutter build web --release --no-web-resources-cdn
```

> 한글 표시를 위해 `assets/fonts/NotoSansKR.ttf`(Noto Sans KR, OFL)를 번들로 포함합니다.
> 외부 폰트 CDN(gstatic)에 의존하지 않으므로 오프라인에서도 한글이 깨지지 않습니다.

## 비고

- 데이터는 모두 데모용 목업이며, 실제 서비스에서는 API 연동이 필요합니다.
- 세액공제액은 기부금 세액공제율(1천만원 이하 15%)을 단순 적용한 추정치입니다.
